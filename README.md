# RotarTrack for Aggie Rotaract

## How to Run Locally

 1. Clone the repository
 2. First run `bundle`
 3. Then create the database `rails db:create`
 4. Migrate, `rails db:migrate`
 5. Finally run `rails server --binding=0.0.0.0`
